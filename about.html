About This Project

This is a school project: a simple Library Management System for a librarian.

Main Features

  • View the books list
  • Add a new book
  • Edit an existing book
  • Store data in a database using SQL

Book Fields

  • Title
  • Author
  • Year
  • Genre
  • Rating
  • Location
  • Barcode
